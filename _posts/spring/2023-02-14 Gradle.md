Gradle은 무엇 인가?


Gradle은 오픈소스 빌드 자동화 도구다.
빌드 자동화 도구는 compile, test, packaging, Deploy & Run 을 제공해준다.

- 특징

1. 높은 성능 : 다야안 캐시를 사용하요 이전 빌드의 출력을 채사용 하고, 공유 빌드 캐시를 사용하면 다른 시스템의 출력을 재사용 할 수도 있어 높은 선능을 낼 수 있다.
2. JVM 기반 : Gradle은 JVM에서 실행된다. 빌드의 로직을 표준 자바 API를 사용하여 빌드 할 수 있다.
3. 확장성 : Gradle은 쉽게 확장하여 사용자 지정 작업 및 플러그인과 함께 사용자 정의 빌드 로직을 제공할 수 있습니다
4. 호환성 : 안드로이드 스튜디오, IntelliJ IDEA, 이클립스, VSCode, NetBeans 를 포함한 몇몇 주요 IDE들이 Gradle과 상호 작용을 지원한다.
5. Build Scan™ : Build Scan™ 을 이용하여 문제를 식별하는 데 사용할 수 있는 빌드에 대한 광범위한 정보를 제공한다.
